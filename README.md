## рҹҢҢ Mission Overview

> *"One packet. One second. Sensor to screen. Three hardware systems, one ground station, zero tolerance for failure."*

**INVICTUS II** is the complete avionics and ground station stack for three UKSEDS competitions:

| Competition | Vehicle | Avionics |
|---|---|---|
| **MachX** | Bigger rocket + CanSat inside | STM32 Bluepill В· RFM69HCW 433MHz |
| **NRC** | Smaller standalone rocket | Heltec LoRa V3 В· LoRa 868 MHz |
| **NRC Rover** | Rocket + Rover deployment (later) | RPi 4B В· Flask В· BTS7960 |

```
Target Altitude  вҶ’  2,200 ft (670 m)
Telemetry Rate   вҶ’  1 Hz (1 packet/second)
Radio Links      вҶ’  433 MHz RFM69 (MachX CanSat) + LoRa 868 MHz (NRC Rocket)
Ground Station   вҶ’  Node.js В· SQLite В· Socket.io В· Chart.js
Rover            вҶ’  Raspberry Pi 4B В· Flask В· BTS7960 В· Camera Module 3
```

## вҡЎ Quick Start

### Prerequisites

```bash
node --version   # v20 LTS required
npm --version    # v9+ required
```

### рҹ–ҘпёҸ Ground Station вҖ” 3 Commands

```bash
# 1. Clone & install
git clone https://github.com/theSaksham02/Invictus-II.git
cd Invictus-II/backend
npm install

# 2. Configure
cp .env.example .env
# вҶ’ Set SERIAL_PORT_CANSAT, SERIAL_PORT_NRC, and 115200 baud values for your OS

# 3. Launch рҹҡҖ
npm start
# вҶ’ http://localhost:3000
```

### рҹ§Ә Simulation Mode (No Hardware Needed)

```bash
npm run sim
# Streams a full parabolic flight at 1Hz
# Open http://localhost:3000 вҖ” watch altitude climb to 670m and back
```

```bash
pip install flask picamera2 RPi.GPIO
python3 firmware/rover/app.py
# вҶ’ Accessible at 192.168.4.1:5000

## рҹ“Ў API Reference

### REST Endpoints

```
GET  /api/health          вҶ’  System status, uptime, signal state
GET  /api/packets         вҶ’  ?source=CANSAT&limit=200&since=0
GET  /api/stats           вҶ’  Max alt, min temp, packet counts per source
GET  /api/export          вҶ’  ?source=CANSAT вҶ’ downloads flight.csv

POST /api/rover/control   вҶ’  { left: 100, right: -100 }

### Socket.io Events

```
вҶҗ packet              Every telemetry packet (1 Hz)
вҶҗ mission_event       IDLEвҶ’LAUNCHEDвҶ’ASCENDINGвҶ’APOGEEвҶ’DESCENDINGвҶ’LANDED
вҶҗ signal_lost         No packet received for 5 seconds
вҶҗ signal_recovered    Signal resumed after gap
вҶҗ sd_upload_complete  SD card CSV processed
вҶҗ history             Sent on connect: last 60 packets + all events

вҶ’ subscribe_source    { source: "CANSAT" | "NRC" | "ALL" }
вҶ’ request_history     { source, limit }
```

## рҹ“Ұ Packet Specification

### CANSAT вҖ” 37-byte Binary (Little-Endian)

```
Offset  Size  Type     Field
в”Җв”Җв”Җв”Җв”Җв”Җ  в”Җв”Җв”Җв”Җ  в”Җв”Җв”Җв”Җв”Җв”Җв”Җ  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
0       2     uint16   pkt_id
2       4     uint32   timestamp_ms
6       4     float32  altitude_m
10      4     float32  temp_c
14      4     float32  pressure_hpa
18      4     float32  accel_z
22      4     float32  gyro_x
26      4     float32  lat
30      4     float32  lon
34      1     int8     rssi_dbm
35      1     uint8    flags  (bit0=launched  bit1=apogee)
36      1     uint8    checksum  (XOR of bytes 0вҖ“35)
```

### NRC Rocket вҖ” ASCII CSV

```
NRC2:<pkt_id>,<timestamp_ms>,<altitude_m>,<temp_c>,<pressure_hpa>,<lat>,<lon>,<rssi_dbm>,<flags>,<crc16_hex>\n
```
